# TC Med Study App

TC Med Study App is a student learning platform designed to help university students prepare for exams through CBT practice, flashcards, study groups, and performance tracking.

## Features

- CBT exams with timer
- Instant scoring
- Leaderboards
- Course-based question banks
- Flashcards
- Study notes sharing
- Student profiles
- Performance analytics

## Target Users

- University students
- Medical students
- Students preparing for CBT examinations

## Tech Stack

- Frontend: To be decided
- Backend: Supabase
- Database: PostgreSQL
- Hosting: Netlify

## Vision

To become the leading study and exam preparation platform for African university students.
